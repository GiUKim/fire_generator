# fire_generator
---
- /roadcar/ 폴더에 데이터셋 삽입 후 fire_generator.py 실행
- 좌클릭 -> 화염 생성
- 1 press -> 화염 커짐 (20%)
- 2 press -> 화염 작아짐 (-20%)
- 우클릭 -> bounding box 생성 후 .txt에 yolo형식으로 저장
- esc -> 종료
